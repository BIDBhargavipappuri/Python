# What This Project Does

•	Takes a city name as input

•	Calls a real weather API

•	Fetches temperature, humidity, wind speed, conditions

•	Shows results in a clean format

•	Logs every search

•	Optional: Sends daily weather email

•	Optional: Saves data to CSV for trend analysis

# Get a Free Weather API Key:

Use OpenWeatherMap (free, no credit card).

1.	Go to: https://openweathermap.org
   
2.	Create a free account
	
3.	Go to API Keys
	
4.	Copy your key

You’ll use it like:

https://api.openweathermap.org/data/2.5/weather?q=London&appid=YOUR_API_KEY
